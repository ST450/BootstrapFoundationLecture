# Bootstrap 基礎講座

Udemyで受講したファイル類になる。


## 画面サイズ
```
extra small（極小）         576px未満
small（小）         - sm    576～767px
medium（中）        - md    768～991px
large（大）         - lg    992～1199px
extra large（特大） - xl    1200px以上
```

開発ツールを表示して画面を縮小、拡大すると右上に表示しているサイズを確認できる。
